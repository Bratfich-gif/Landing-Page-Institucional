## Sobre o Projeto
Este projeto foi criado para apresentar serviços de tecnologia de uma empresa fictícia, de forma moderna e profissional.

## Funcionalidades
- **Design Responsivo**: Adaptável para mobile, tablet e desktop
- **Animações Suaves**: Efeitos de scroll e hover para melhor interatividade
- **Navegação Fixa**: Menu que acompanha o scroll do usuário
- **Scroll Suave**: Navegação entre seções com animação
- **Formulário de Contato**: Seção para captação de leads
- **Cards Interativos**: Apresentação de serviços com efeitos visuais

## Seções

1. **Início**: Apresentação principal
2. **Serviços**: Três cards destacando os principais serviços
3. **Sobre**: Informações da empresa
4. **Contato**: Formulário para captação de leads
5. **Footer**: Informações de contato e redes sociais

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilização e animações
- **JavaScript**: Interatividade e efeitos dinâmicos
- **Tailwind CSS**: Framework CSS utility-first
- **Google Fonts**: Tipografia (Inter)

## Instalação
1. Clone este repositório: ```git clone https://github.com/Bratfich-gif/Landing-Page-FullStack.git```
2. Navegue até o diretório do projeto: ```cd Landing Page```
3. Abra o arquivo `index.html` no seu navegador ou utilize uma extensão como Live Server no Visual Studio Code (VS Code).

## Personalização
### Cores
Para alterar o esquema de cores, modifique as classes do Tailwind ou o gradiente no CSS:
```.gradient-bg {background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);}```

### Conteúdo
Edite o arquivo index.html para personalizar:

- Textos e títulos
- Imagem
- Links de redes sociais
- Informações de contato

### Tipografia
Para alterar a fonte, substitua a importação do Google Fonts e atualize a propriedade font-family no CSS:
```body {font-family: 'Sua-Fonte', sans-serif;}```
